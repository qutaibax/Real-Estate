# Real Estate Project

## Description
This is a simple real estate management project.
It allows users to manage property listings.
The project was built for learning purposes.

## Technologies Used
- Laravel
- PHP
- MySQL


## Features
- Authentication 
- Verify code (otp)
- Admin controls register requests 
- Add apartment
- Edit apartment
- View apartment
- Book apartment
- Edit bookings
- Cancel bookings
- Browse all bookings 
- Filter apartment
- Rate apartment
- Owner controls bookings

## What I Learned
- CRUD operations
- Database integration
